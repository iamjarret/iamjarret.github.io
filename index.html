Pystockfish!
Pystockfish is a python package that integrates the world's strongest open-source chess engine into Python. I built it in a hotel room in Morocco, and today it has 500+ downloads/month and an active online development community. It was the subject of a presentation at Pygotham 2014, "Helping Python Play Chess."
